# Usage

Press arrows or swipe left, right, up or down to move tiles. Try to score 2048 to win. Press restart to try again.

# Installation

1. Clone repository 
```bash
git clone https://github.com/Vasya-556/2048
```
2. Go into project folder
```bash
cd 2048
```

## Manual
3. Install dependencies
```bash
npm install
```
4. Build
```bash
npx tsc
```
5. Start server
```bash
npm run start
```

## With script
3. Build project
```bash
./server.sh -b
```
4. Start server
```bash
./server.sh -s
```
